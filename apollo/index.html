apollo

Overview

Apollo is a mobile-first, global css framework with pre-styled HTML elements, an extensive column-based gridwork, and helper classes. This framework is currently developed by Joshua Potter.

Apollo utilizes normalize.css, by Nicolas Gallagher and Jonathan Neal.

Getting Started

In order to make your document Apollo compliant, you must first define a few development properties.

HTML5 Doctype

<!doctype html>
<html>
...
</html>

Viewport

The viewport allows for mobile rendering of the webpage. Add this to the <head> tag. 

<meta name="viewport" content="width=device-width, initial-scale=1">

Link colors

If you do not want to use the default link colors designated by the user's web browser, you must define your base link color in your css stylesheet. 

a {
    color: #3d3d3d;
}

Containers

Apollo takes advantage of containers to wrap site content with 15px of horizontal padding. These containers act as the wrappers for the gridwork system, as well. Class .container is fixed width. 

<div class="container">
...
</div>

Class .container-fluid spans the full-width of your document. 

<div class="container-fluid">
...
</div>

Gridwork System

Apollo's pride and joy: the gridwork system. The gridwork system allows the developer to stack elements side-by-side with perfect alignment and positioning, having them resize perfectly for mobile.

How It Works

In order to use the gridwork system, the developer creates horizontal rows. The class .row must be placed within a container class. Within the .row class, place your columns. There is a negative horizontal margin to account for positions of columns, which have horizontal padding.

There are 12 column types. An example of a column construction would be as follows: 

<div class="row">
	<div class="col-r-6">.col-r-6</div>
	<div class="col-r-6">.col-r-6</div>
</div>
<div class="row">
	<div class="col-r-4">.col-r-4</div>
	<div class="col-r-4">.col-r-4</div>
	<div class="col-r-4">.col-r-4</div>
</div>
<div class="row">
	<div class="col-r-3">.col-r-3</div>
	<div class="col-r-3">.col-r-3</div>
	<div class="col-r-3">.col-r-3</div>
	<div class="col-r-3">.col-r-3</div>
</div>

Class constructions consists of "col-" for column, "r-" for responsive ("nr-" for non-responsive versions that do not break down on mobile responsive state), and then a number for the width of the container. These numbers range from 1 through 12 and stand for the number of containers wide it should be. Below is an example of the code above (styled for emphasis):

.col-r-6
.col-r-6
.col-r-4
.col-r-4
.col-r-4
.col-r-3
.col-r-3

.col-r-3

Typography

Apollo does not apply a specific font, but rather uses the default sans-serif typeface of the user's computer. You may apply your own specific font by defining it in your own stylesheet.

Headings

Apollo provides header stylings for <h1> - <h4> with a font-weight: 700;. Here are some examples:

h1 - Heading

font-size: 36px;

h2 - Heading

font-size: 28px;

h3 - Heading

font-size: 22px;

h4 - Heading

font-size: 18px;
